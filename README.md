#Twitter Sentiment Analysis

#📌 Overview

This project performs sentiment analysis on Twitter data to classify tweets as positive or negative using Natural Language Processing (NLP) and Machine Learning techniques.

#🚀 Features
🧹 Data cleaning and preprocessing of tweets
🔤 Text transformation using TF-IDF Vectorization
🤖 Sentiment classification using Logistic Regression
📊 Model evaluation using accuracy metrics

#🧠 How It Works
Tweet → Cleaning → Stemming → TF-IDF → Model → Sentiment Prediction
 Data Cleaning
  Removes URLs, mentions, special characters
  Converts text to lowercase
 Text Preprocessing
  Applies stemming to normalize words
 Feature Extraction
  Converts text into numerical features using TF-IDF
 Model Training
  Uses Logistic Regression to learn sentiment patterns
 Prediction
  Classifies tweets as Positive or Negative
